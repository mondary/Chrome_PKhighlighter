Politique de confidentialite

PKhighlighter Chrome Extension

Derniere mise a jour : 26 juin 2026

Cette page est la politique de confidentialite officielle de l'extension Chrome PKhighlighter. This page is the official privacy policy for the PKhighlighter Chrome extension. An English version is available below the French version.

Donnees traitees

PKhighlighter ne collecte pas de donnees personnelles pour un serveur externe. L'extension traite uniquement les informations necessaires pour surligner des mots-cles, barrer des termes exclus et memoriser vos preferences par site.

  • Les mots ou phrases que vous ajoutez dans les listes de surlignage et d'exclusion ;
  • Le style visuel choisi pour le surlignage ;
  • Les noms de domaine sur lesquels l'extension est active ;
  • Le texte visible des pages web, traite localement pour appliquer le surlignage.

Utilisation des donnees

Ces donnees sont utilisees uniquement pour appliquer le surlignage et les exclusions sur les pages web, afficher l'interface de configuration et restaurer vos preferences sur chaque site concerne.

Stockage des donnees

Les preferences de l'extension sont stockees localement dans votre navigateur, principalement via window.localStorage, avec des cles dediees au domaine courant. Aucune synchronisation avec un serveur externe n'est effectuee.

Partage et transfert

PKhighlighter ne vend pas, ne loue pas, ne partage pas et ne transfere pas les donnees utilisateur a des tiers. Les donnees ne sont pas utilisees pour la publicite, le suivi analytique, le profilage, la solvabilite ou des finalites sans rapport avec la fonctionnalite de base de l'extension.

Cookies et suivi

L'extension n'utilise pas de cookies, de pixels de suivi, d'outils analytiques ou de technologies publicitaires.

Acces et suppression

Vous pouvez modifier ou supprimer vos mots-cles et preferences depuis l'interface de l'extension. Vous pouvez aussi effacer les donnees locales du navigateur pour retirer toute configuration enregistree par PKhighlighter.

Permissions Chrome

Les permissions et scripts declares servent uniquement a injecter localement la logique de surlignage sur les pages web, memoriser vos reglages et permettre l'ouverture de l'interface flottante de configuration.

Modifications de cette politique

Cette politique peut etre mise a jour si les fonctionnalites de l'extension changent. La date de mise a jour sera modifiee sur cette page.

Contact

Pour toute question concernant cette politique de confidentialite, contactez : GitHub Issues.

English Version

Data Processed

PKhighlighter does not collect personal data for an external server. The extension only processes the information required to highlight keywords, strike excluded terms, and remember your preferences per site.

  • The words or phrases you add to the highlight and exclusion lists;
  • The visual style selected for highlighting;
  • The domain names on which the extension is enabled;
  • The visible text of web pages, processed locally to apply highlighting.

Use of Data

This data is used only to apply highlighting and exclusions on web pages, display the configuration interface, and restore your preferences on each relevant site.

Data Storage

Extension preferences are stored locally in your browser, mainly via window.localStorage, using keys scoped to the current domain. No synchronization with an external server is performed.

Sharing and Transfer

PKhighlighter does not sell, rent, share, or transfer user data to third parties. Data is not used for advertising, analytics tracking, profiling, creditworthiness, or purposes unrelated to the extension's core functionality.

Cookies and Tracking

The extension does not use cookies, tracking pixels, analytics tools, or advertising technologies.

Access and Deletion

You can modify or delete your keywords and preferences from the extension interface. You can also clear your browser's local data to remove all configuration saved by PKhighlighter.

Chrome Permissions

The declared permissions and scripts are used only to inject the highlighting logic locally on web pages, remember your settings, and allow the floating configuration interface to open.

Changes to This Policy

This policy may be updated if the extension features change. The update date will be changed on this page.

Contact

For questions about this privacy policy, contact: GitHub Issues.
