# PK Highlighter - Description Chrome Web Store

## 🇬🇧 English

Highlight keywords with custom colors. Made by PK-Labs.

Powerful keyword highlighting tool for research and content analysis. Highlight important terms, exclude irrelevant phrases, and customize visual styles.

**Features:**
- Highlight keywords with 5 visual styles (Normal, Bold, Origami, Candy, Sticker)
- Strike-through excluded terms in red
- Per-site settings persistence
- Works on all websites
- Instant application on page load

Perfect for researchers, content analysts, and anyone who needs to quickly identify important information while browsing.

- Fonctionne sur tous les sites web
- Application instantanée au chargement de la page

Parfait pour les chercheurs, analystes de contenu et toute personne qui a besoin d'identifier rapidement les informations importantes pendant la navigation.
