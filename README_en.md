# PK Keyword Highlighter 🖍️

![Project icon](icon.png)

[🇬🇧 EN](README_en.md) · [🇫🇷 FR](README.md)

✨ A userscript that highlights keywords and strikes through excluded phrases, with multiple visual styles.

## ✅ Features
- **Automatic Highlighting** : Immediate visual emphasis of keywords.
- **Smart Exclusion** : Strikes through excluded phrases in red to prevent false positives.
- **Visual Styles** : Choose between Normal, Bold, Origami, Candy, and Sticker.
- **Local Persistence** : Per-site settings saved via local storage.

## 🧠 Usage
1. Install a userscript manager (Tampermonkey, Violentmonkey).
2. Add the `keyword-highlighter.user.js` script.
3. Click the **HL** button on your page to open the configuration panel.
4. Configure your lists:
   - **Highlight**: words/phrases to highlight (comma-separated).
   - **Exclude**: words/phrases to strike through (comma-separated).
   - **Style**: choose the visual rendering.
5. Click **Save** to apply.

## ⚙️ Settings
- Exclusions are applied before highlighting.
- The script ignores input fields and its own UI panel.
